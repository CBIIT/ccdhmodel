Class: SpecimenCreationActivity

The process of creating a specimen through observing and/or removing material from an biological source or natural setting (e.g. from a person, animal, cell line, or environmental material), or through derivation from an existing specimen (e.g. through portioning or aliquoting). This activity represents the entire process up to the point where the specimen is physically modified, stored, or transported.

URI: ccdh:SpecimenCreationActivity

Attributes

Own

  • ➞activity_type OPT
    • Description: The high-level type of creation activity
    • range: String
  • ➞additive 0..*
    • Description: A type of material or reagent used as input when creating the specimen
    • range: String
  • ➞collection_method_type OPT
    • Description: A type of method followed in collecting a sample form its original source.
    • range: String
  • ➞collection_site OPT
    • Description: A data container holding information about the anatomic source of a specimen, including its location in the body, and the approach route used to access it.
    • range: String

  • ➞date_ended OPT

    • Description: The date when the activity ended (in this case, a specimen creation event).

    • range: String

    • ➞date_started OPT

    • Description: The date when the activity began (in this case, a specimen creation event).

    • range: String

    • ➞derivation_method_type OPT
    • Description: A type of method followed in deriving new specimens from an existing one.
    • range: String
    • ➞execution_condition 0..*
    • Description: Information describing the specified or observed conditions under which activity was executed
    • range: String
    • ➞input_specimen 0..*
    • Description: One or more existing specimens input into the specimen creation activity, from which a new specimen is derived (e.g. through portioning, aliquoting, sectioning, etc)
    • range: String
    • ➞performed_by OPT
    • Description: The organization or group that performed the activity.
    • range: String
    • ➞quantity_collected OPT
    • Description: The quantity of material in the specimen as originally collected from its original source material (prior to downstream portioning or processing)
    • range: String
    • ➞specimen_order OPT
    • Description: A numeric value that represents the sequential order of this creation activity relative to that of creation of other specimens from the same source material or parent specimen.
    • range: String
