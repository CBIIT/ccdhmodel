Class: SpecimenTransportActivity

An activity through which a specimen is transported between locations.

URI: ccdh:SpecimenTransportActivity

Parents

  • is_a: Entity - Any resource that has its own identifier

Attributes

Own

  • ➞date_ended OPT
    • Description: The date when the activity ended (in this case, a transport event).
    • range: CcdhDateTime
  • ➞date_started OPT
    • Description: The date when the activity began (in this case, a transport event).
    • range: CcdhDateTime
  • ➞execution_conditions 0..*
    • Description: An environmental condition under which the activity is executed (in this case, a transport activity).
    • range: CcdhString
  • ➞transport_destination OPT
    • Description: An organization (facility, site, lab, etc) to which the specimen is delivered as a result of the activity.
    • range: Organization

Inherited from Entity:

  • ➞id OPT
    • Description: The 'logical' identifier of the entity in the system of record, e.g. a UUID. This 'id' is unique within a given system of record. The identified entity (with same "identifier" value for example) may have a different 'id' in a different system.
    • range: CcdhString
