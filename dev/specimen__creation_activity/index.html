Slot: specimen__creation_activity

An activity or procedure through which a specimen was created, through removing material from an biological subject, or deriving material from an existing specimen.

URI: ccdh:specimen__creation_activity

Domain and Range

None -> 0..1 SpecimenCreationActivity

Parents

Children

Used by

  • Specimen

Other properties
Comments: Note that a given specimen is the result of only one and only one SpecimenCreationActivity, but may be subject to many processing events that modify or derive new speciemns from it.
